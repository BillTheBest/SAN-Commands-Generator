##############
Name : SAN Commands Generator
####################
Compatibility : The application provided the needed commands comatibile with Brocade and Cisco switches. 
####################
How to use:
- You need to use the Excel templates provided within the application folder
- Use whatever templates "Cisco or Brocade"
- Open the application.
- Choose the the type of switch, the excel input file, the output directory, and the outputfile.txt.
- Open the output configuration file with notpad++ or similar applications.
########################
Note :
- The output configuration file assume that the switch is never been configured and creates everything from the beginning.
- You might just need part of the configuration file and not everything. 
- You always need to test and review the output configuration file before implementation on the production environment.
